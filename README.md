# TIE eLibrary

Access hundreds of TIE textbooks online — anytime, anywhere. Read Pre-Primary to Form 6 books for free.

## Features

- Browse books by type, level, and class
- Flipbook reader for online textbooks
- User authentication (admin & regular users)
- Book borrowing system
- Admin panel for user management

## Deployment

This is a Docker-based Java application that runs on port 7860.
